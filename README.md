# Kara for p5js
A Kara implementation for [processing](https://processing.org/) on [p5js](https://p5js.org/).

## Examples
[Simple Example](https://editor.p5js.org/tkilla77/full/NdMUAvfdD)

```js
function my_kara(kara) {
  kara.move();
  kara.move();
  kara.turnRight();
  kara.move();
  kara.move();
  kara.putLeaf();
}
```
